# FirstFlip Exchange UI

Modern, premium crypto trading platform UI built with React.

## Features
- 🎨 Beautiful dark theme design
- 📱 Mobile-first responsive UI
- ⚡ Interactive components (FAB, cards, modals)
- 🔄 Multiple app modes (Exchange, Flip, Falcon Wallet)
- 💳 Crypto asset management
- 📊 Live rate updates

## Quick Start

### Option 1: CodeSandbox (Recommended)
1. Go to [codesandbox.io](https://codesandbox.io)
2. Click "Import Repository"
3. Paste: `https://github.com/firstflipofficial-lab/firstflip-ui`
4. Done! 🚀

### Option 2: Local Setup
```bash
git clone https://github.com/firstflipofficial-lab/firstflip-ui.git
cd firstflip-ui
npm install
npm start
```

## Components
- **FirstFlipUI** - Main app container with app switcher
- **ExchangeHome** - Home screen with portfolio & live rates
- **TradeScreen** - Buy crypto interface
- **BusinessScreen** - Business mode with QR code generation
- **FlipXScreen** - FLIPX token management
- **CardScreen** - Virtual card display with flip animation
- **ProfileScreen** - User profile & settings
- **FlipApp** - Gift card & utilities platform
- **FalconWallet** - Web3 wallet interface

## Design System
Custom aqua/cyan theme with glassmorphism effects and smooth animations.
